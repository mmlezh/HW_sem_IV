\doxysubsection{Detailed Description}
Contains the definition of the myjson class for handling JSON data. 

This class provides methods for reading data from a file or stream in JSON format, manipulating the data, and writing the data back out to a file or stream in JSON format.

The class uses the nlohmann\+::json library to handle the JSON data.
